Survive To The Infinty:

Is a RogueLike inspired by the binding of isaac and magic survival with a Wild West aesthetic.

The goal is simple, try to survive as long as possible and collect as many points as possible.

---------------------------------------------------------------------------------------------------

Controls:

WASD :      MOVE

Up  -  Left  -  Down  -  Right :       SHOOT
